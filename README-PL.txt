ANNA POŹNIAK — GABINET PSYCHOLOGICZNY

To jest kompletna polska wersja strony z dołączonymi obrazami, ikonami i stroną blogową.

PLIKI:
- index.html — strona główna
- blog.html — blog
- css/style.css — wygląd strony
- js/script.js — menu mobilne i animacje

2. Otwórz plik index.html w przeglądarce.
